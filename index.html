Getting started

PHP School is a set of ever expanding workshops to teach you basic to advanced concepts in PHP.

composer global require php-school/learn-you-php

Community workshops

At the core, PHP School is a workshop application which allows you to run exercises. We have designed it in a way so that you can build your own workshops!

If you want your workshop featured here, drop us an issue on GitHub. Visit the documentation to see how you can build your own workshop!

Need help?

If you need help or want to contribute to PHP School. You can chat on Slack, Create issues and send PR's on GitHub. We also have a twitter account where we will push out news and information!

Our team

We are a couple of friends working in our spare time to improve the quality of PHP learning resources. We want to empower you to build, whether it be a hobbyist project or a tool to help educate the world. Skill nor age should be a barrier to learning, the PHP School team are committed to helping train the next generation of coders who will go on to change the world.
